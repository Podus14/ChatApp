# Project Setup Guide

This project contains two parts: **server** and **client**.  
Follow the steps below to install dependencies and run the application.

## Installation

### Install Dependencies

#### Server

```bash
cd server
npm install
```

To build the server:

```bash
npm run build
```

To start the server:

```bash
npm run start
```

#### Client

```bash
cd client
npm install
```

To start the client:

```bash
npm run dev
```
